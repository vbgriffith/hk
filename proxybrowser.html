PROXR
In-Browser Proxy Navigator & Network Probe Tool
🌐
Proxy Browser
Browse websites through multiple CORS proxy services. Bypass same-origin restrictions and access content from any domain using AllOrigins, CORS Proxy, ThingProxy, or a custom proxy URL.
🔍
Port Scanner
HTTP-based port probing across any target host. Detect open and accessible ports with configurable timeout, concurrency, and preset port ranges. Real-time results with service identification.
🛡️
Risk Analysis
Each open port is automatically analyzed and tagged with a risk level based on the service running. Identify potential security exposures like open databases, remote desktop, or unencrypted services.
📊
Export & Reporting
Export scan results as JSON for use in security audits, network documentation, or integration with other tools. Real-time scan log with timestamped entries tracks every probe attempt.
