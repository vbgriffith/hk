What to do with results
If Cockpit is found at port 9090 — open it for a full browser-based terminal and file manager.
If Jupyter is found — you have a terminal and can run python3 -m http.server 8000 to serve files to the whole network.
If Apache / Nginx is found on port 80 — files in /var/www/html are accessible to everyone on the LAN.
If nothing is found — the network may have a firewall blocking cross-device connections, or servers may be running on non-standard ports.
