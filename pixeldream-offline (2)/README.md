# PixelDream — 100% Offline Text-to-Image Generator

> Zero downloads. Zero internet. Zero setup. Double-click `index.html` and generate.

---

## 🚀 Quick Start

1. Unzip the folder
2. Double-click **`index.html`**
3. Type a prompt → click **Generate Image**

That's it. No server, no install, no internet, no loading screen.

---

## ✅ Truly Self-Contained

`index.html` has **zero external dependencies**:

| Dependency | Status |
|---|---|
| Model weights | ✅ Embedded as JS constants |
| Fonts | ✅ System fonts (Palatino / monospace) |
| CDN libraries | ✅ None used |
| Service Worker | ✅ Not needed |
| Internet connection | ✅ Never needed |

Works from: `file://` (double-click), USB drive, network share, air-gapped machine.

---

## 🎨 Features

- 14 subject types — humans (baby → elderly) + animals
- 12 environments — beach, forest, snow, urban, lab, sea, rooftop…
- 8 lighting modes — sunset, moonlit, candlelight, dramatic…
- 6 style filters — Photorealistic, Oil Painting, Watercolour, Pencil, Impressionist, Neon
- 4 output sizes — 256 / 512 / 640 / 768 px
- Seeded generation — same prompt + seed = same image every time
- Gallery + PNG download
