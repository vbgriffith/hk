## 1. Kalamax, the Stormsire

**Colors:** Temur (Red/Green/Blue)
**Core Mechanic:** Kalamax copies the FIRST instant you cast each turn if he is already tapped. Each copy places a +1/+1 counter on him. Keep Kalamax tapped, chain spells, and scale into overwhelming copy loops or an unblockable 21-damage commander.

### Strategy Pillars

| Pillar | Goal |
|---|---|
| **Tap Enablers** | Get Kalamax tapped safely before casting key spells |
| **Fork / Copy Effects** | Multiply how many times each spell resolves |
| **Storm / Value Chain** | Cast cheap instants repeatedly to grow Kalamax exponentially |
| **Win Conditions** | Infinite mana kill, commander damage, or storm finish |

### Win Condition 1: Infinite Mana + Fireball

**Required pieces:** Kalamax (tapped) + Isochron Scepter (Dramatic Reversal imprinted) + 3+ mana worth of non-land mana sources

| Step | Action |
|---|---|
| 1 | Cast any cantrip or attack to get **Kalamax tapped** before your main phase |
| 2 | Activate **Isochron Scepter** — cast Dramatic Reversal (imprinted); Kalamax copies it because it's an instant |
| 3 | **Dramatic Reversal** resolves — untap all non-land permanents including all mana rocks and Kalamax |
| 4 | **Kalamax's copy** also resolves — untaps everything again (double untap); you have more mana than you started |
| 5 | Activate Isochron Scepter again — Kalamax copies Dramatic Reversal again; each loop nets +1 mana with 3+ rocks |
| 6 | Repeat until you have **infinite mana** in your mana pool |
| 7 | Cast **Fireball** or **Banefire** targeting all three opponents; with infinite mana deal lethal damage to all |
| 8 | Alternative: Cast **Blue Sun's Zenith** targeting each opponent to mill them out with infinite mana |

### Win Condition 2: Commander Damage (Combat)

**Required pieces:** Kalamax + Fork effects + pump instants

| Step | Action |
|---|---|
| 1 | Early game: attack with Kalamax every turn to tap him, copying instants and accumulating +1/+1 counters |
| 2 | Cast **Bonus Round** (2RR) — now every instant you cast this turn is copied once extra beyond Kalamax's trigger |
| 3 | Cast **Reverberate** targeting your next instant — with Bonus Round and Kalamax you now get 4 copies of each spell |
| 4 | Stack pump spells: **Berserk** (G) targeting Kalamax — with 4 copies each granting +X/+0 and trample, Kalamax becomes enormous |
| 5 | Cast **Savage Beating** — give Kalamax double strike and untap creatures, enabling a second attack this turn |
| 6 | Attack with Kalamax — double strike means two hits; each hit counts toward 21 commander damage |
| 7 | With 10+ counters from previous turns + pump spells, **21 commander damage** in a single attack is achievable |
| 8 | Cast **Temur Battle Rage** as backup — give Kalamax double strike and trample through any blocker |

### Win Condition 3: Storm Finish (Grapeshot / Brain Freeze)

**Required pieces:** Kalamax (tapped) + Reiterate + mana engine + Grapeshot or Brain Freeze

| Step | Action |
|---|---|
| 1 | Ensure Kalamax is tapped. Have 6+ mana available |
| 2 | Cast **Bonus Round** (2RR) — all instants/sorceries are copied for the rest of the turn |
| 3 | Cast a **mana-generating instant** like Desperate Ritual (adds 3R) — with Kalamax copy and Bonus Round copy, generate net mana |
| 4 | Cast **Reiterate** (1RR) targeting the Desperate Ritual — copy it; pay Reiterate's Buyback cost (2RR) to return it to hand |
| 5 | Kalamax copies Reiterate (places +1/+1 counter); Bonus Round copies Reiterate again — storm count climbs fast |
| 6 | Repeat the Reiterate loop, generating more mana each cycle and incrementing storm count; goal is 20+ storm count |
| 7 | Cast **Grapeshot** — storm triggers for each previous spell; with 20 storm copies, deal 20 damage to each opponent |
| 8 | With Kalamax copy + Bonus Round, Grapeshot produces 40+ damage total — enough to kill the table |
| 9 | Alternative: **Brain Freeze** mills 3 per storm copy; with 20+ copies, each opponent mills 60+ cards = lethal mill |

## 2. Aminatou, the Fateshifter

**Colors:** Esper (White/Blue/Black)
**Core Mechanic:** Aminatou's -1 blinks any permanent you control, resetting ETB triggers. Her +1 manipulates the top of your library. Her -6 redistributes all permanents among players — a game-winning ultimate when your board is better than everyone else's.

### Strategy Pillars

| Pillar | Goal |
|---|---|
| **ETB Value Engines** | Stack creatures/artifacts that trigger powerfully when they enter |
| **Proliferate** | Accelerate Aminatou's loyalty and spread counters on planeswalkers |
| **Top-Deck Manipulation** | Use Aminatou's +1 to set up perfect draws each turn |
| **Ultimate Setup** | Engineer a board state where Aminatou's -6 wins the game |

### Win Condition 1: Felidar Guardian Infinite Loop

**Required:** Aminatou on battlefield at 2+ loyalty + Felidar Guardian + any ETB value engine

| Step | Action |
|---|---|
| 1 | Have **Aminatou** (2+ loyalty) and **Felidar Guardian** both on the battlefield |
| 2 | Use **Aminatou's -1** — target and flicker Felidar Guardian (exile it, return immediately) |
| 3 | Felidar Guardian's **ETB triggers** — choose to flicker Aminatou (exile Aminatou, return to battlefield) |
| 4 | Aminatou re-enters with **3 loyalty** (she resets each time she ETBs); she is back to full loyalty |
| 5 | Use **Aminatou's -1 again** — flicker Felidar Guardian; Guardian ETB triggers again, flickers Aminatou again |
| 6 | This loop is **infinite** at zero mana cost once both are on the field |
| 7 | Each loop cycle triggers other ETB cards: **Mulldrifter** = draw 2 per loop; **Peregrine Drake** = untap 5 lands per loop |
| 8 | With **Peregrine Drake** in loop: after 3 loops you have untapped 15 lands — effectively infinite mana |
| 9 | With **Panharmonicon** in play: each ETB triggers twice = double all loop benefits |
| 10 | End the loop by casting **Thassa's Oracle** (win if devotion >= cards in library) or **Jace** mill finish |

## 3. Nevinyrral, Urborg Tyrant

**Colors:** Esper (White/Blue/Black)
**Core Mechanic:** When Nevinyrral dies, pay {B}{U}{W} to destroy ALL creatures and artifacts. He re-enters with Ward protection equal to graveyard creature/artifact count. His death creates a Zombie token for each creature/artifact card in ALL graveyards. The loop: cast, sacrifice, board wipe, rebuild larger each time.

### Strategy Pillars

| Pillar | Goal |
|---|---|
| **Sac Outlets** | Ensure Nevinyrral triggers on demand without waiting to be removed |
| **Graveyard Recursion** | Fill and exploit graveyards continuously |
| **Zombie Tribal Synergy** | Death trigger creates Zombie tokens equal to graveyard count |
| **Board Control** | Wipe repeatedly and tax opponents with Ward protection |

### Win Condition 1: Infinite Zombie Loop (Gravecrawler)

**Required:** Gravecrawler + Phyrexian Altar + any Zombie on board + Diregraf Captain or Blood Artist

| Step | Action |
|---|---|
| 1 | Have **Gravecrawler**, **Phyrexian Altar**, **Diregraf Captain** (or Blood Artist), and one other Zombie on the battlefield |
| 2 | Sacrifice **Gravecrawler** to Phyrexian Altar — produce 1 black mana; Diregraf Captain triggers: target opponent loses 1 life |
| 3 | Gravecrawler is in the graveyard; because you control another Zombie, **cast Gravecrawler from the graveyard** for B |
| 4 | Spend the 1 black mana from Phyrexian Altar to recast Gravecrawler; it re-enters the battlefield |
| 5 | Gravecrawler ETB triggers **Diregraf Colossus** if in play: create another Zombie token |
| 6 | Repeat: sacrifice Gravecrawler again; Diregraf Captain drains 1 life again; recast from graveyard |
| 7 | This is a **zero-net-cost infinite loop**: each cycle costs 1B (Phyrexian Altar produces 1B) = free repeats |
| 8 | With **Rooftop Storm** in play: Gravecrawler costs 0 to cast, making the loop produce **net positive** mana each cycle |
| 9 | Each loop cycle: opponent loses 1 life (Captain) + death/ETB triggers from other cards in play |
| 10 | After enough loops, all opponents are at 0 life; **you win by life loss drain** |

## 4. Sen Triplets

**Colors:** Esper (White/Blue/Black)
**Core Mechanic:** At your upkeep, choose an opponent — until your next turn they cannot cast spells and you may cast spells from their hand using their lands for mana. Sen Triplets must survive to your next upkeep. Protection is everything.

| **Oppressive Locks** | Slow opponents down while you exploit their resources |

### Win Condition 1: Mindslaver Lock (Full Table Control)

**Required:** Mindslaver + Academy Ruins + Sen Triplets on field + 8+ mana available

| Step | Action |
|---|---|
| 1 | Cast **Mindslaver** (6); then pay 4 mana to activate, sacrificing it; gain control of target player's next turn |
| 2 | On that player's turn: cast all their best spells against other opponents; activate abilities; make terrible decisions for them |
| 3 | Mindslaver goes to your graveyard after use |
| 4 | During your turn, tap **Academy Ruins** plus pay U — put Mindslaver from your graveyard on top of your library |
| 5 | Draw Mindslaver on your next draw step; recast for 6; activate again for 4; control that player's turn again |

| 7 | Meanwhile, **Sen Triplets upkeep trigger** selects a DIFFERENT opponent each turn to steal their hand |

### Win Condition 2: Extra Turn Chain + Total Lock

| Step | Action |
|---|---|
| 1 | Cast **Grand Abolisher** (WW) — during your turns, opponents cannot cast spells or activate non-mana abilities |
| 2 | Cast **Teferi, Time Raveler** — opponents can only cast sorceries; they cannot respond at instant speed |
| 3 | Use Sen Triplets upkeep: steal the opponent with the most valuable hand (revealed via **Telepathy**) |

**Required:** Hapatra + Nest of Scarabs + Yawgmoth + sac outlet

| Step | Action |
|---|---|
| 1 | Cast **Hapatra, Vizier of Poisons** and **Nest of Scarabs** — both create tokens whenever -1/-1 counters are placed on any creature |
| 2 | Cast **Yawgmoth, Thran Physician** — pay 1 life + sacrifice a creature: put a -1/-1 counter on any creature + draw a card |
| 3 | Sacrifice any creature to Yawgmoth; target any creature; draw a card |
| 4 | Hapatra triggers: create a 1/1 deathtouch Snake; Nest of Scarabs triggers: create a 1/1 Insect — **2 tokens per activation** |
| 5 | Sacrifice one of those new tokens to Yawgmoth: spread another -1/-1 counter; draw another card; create 2 more tokens |
| 6 | Each activation of Yawgmoth creates **net positive tokens** as long as Hapatra and Nest are both in play |
| 7 | Keep looping: sacrifice tokens, draw cards, create more tokens — effectively infinite token generation and card draw |
| 8 | With 20-30 Snakes and Insects, attack with the horde — Snakes have deathtouch, killing any blocker regardless of toughness |
| 9 | Cast **Craterhoof Behemoth** for the instant win from the token army (pump each creature by creature count) |
| 10 | Alternatively: use Yawgmoth to keep drawing cards until you find and cast **Torment of Hailfire** (XBB) for X=20 |

### Game Plan Decision Framework

For every deck, follow this decision tree each game:

1. **Turn 1-2 — Ramp first.** Sol Ring, Arcane Signet, and color fixing always before your strategy cards
2. **Turn 2-3 — Deploy your commander ONLY when you have protection ready** (Lightning Greaves, etc.) OR when the payoff is immediate
3. **Turn 3-4 — Begin assembling the first required piece** of your primary win condition; tutor if needed
4. **Turn 4-5 — Identify which opponent is closest to winning;** prioritize interaction to slow them while building
5. **Turn 5-6 — Execute your win condition** if all pieces are assembled; hold up counterspell mana when possible
6. **Backup Plan** — If primary combo is disrupted, switch to secondary win condition immediately
7. **Resource Assessment** — Count opponent hands; a player with 0 cards in hand is least threatening; target the player drawing the most
