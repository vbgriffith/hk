Maestro Studio — Feature Roadmap

● = implemented   ○ = planned / suggested next step

Core Notation

  • Piano grand staff (treble + bass)
  • Percussion staff w/ correct noteheads
  • Note durations: whole → 32nd, rests, dotted
  • Accidentals: ♯ ♭ ♮ ♯♯ ♭♭
  • Ties (piano), Beaming (auto)
  • Key & time signatures
  • Multiple bars, undo/delete
  • Tuplets (triplets, quintuplets)
  • Slurs (curved phrase lines)
  • Articulations: staccato, accent, tenuto
  • Fermatas & breath marks
  • Grace notes / acciaccatura
  • Trills & ornaments

Chords & Harmony

  • Manual chord builder (keyboard queue)
  • Named chord library (18 types)
  • Chord inversions (root, 1st, 2nd, 3rd)
  • Chord symbols above staff (e.g. Cmaj7)
  • Roman numeral analysis layer
  • Nashville number system
  • Auto-voice-leading suggestions

Structure & Form

  • Repeat barlines (start/end repeat)
  • 1st & 2nd endings (voltas)
  • D.C. / D.S. / Fine / Coda symbols
  • Rehearsal marks (boxed letters A, B…)
  • Section labels / tempo markings text
  • Multi-page layout with auto-flow
  • System breaks & page breaks

Text & Expression

  • Lyrics under treble staff
  • Tempo markings (Allegro, Adagio…)
  • Expression text (cresc., dim., rall.)
  • Hairpin crescendo / decrescendo
  • Pedal marks for piano
  • Text boxes / composer annotations

Multiple Parts

  • Piano + Percussion layout
  • Add/remove instrument staves
  • Guitar tab staff
  • Chord chart view
  • Part extraction (hide other parts)
  • Transposing instruments (Bb clarinet…)

Playback & Audio

  • MIDI playback via Web Audio / Tone.js
  • Metronome click track
  • Instrument sound fonts (piano, drums)
  • Playhead cursor during playback
  • Loop selection
  • Tempo tap / BPM detection

Import / Export

  • PDF export
  • MusicXML import (Finale/Sibelius)
  • MusicXML export
  • MIDI file export
  • ABC notation import
  • PNG / SVG image export per page

UX / Workflow

  • Click-to-add notes on staff
  • Click-to-select & edit
  • Undo history (60 steps)
  • Copy / paste measures
  • Drag notes to reposition
  • MIDI keyboard input (WebMIDI)
  • Auto-save to localStorage
  • Score templates (lead sheet, drum chart)
